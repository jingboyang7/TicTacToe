# TicTacToe
A simple solution for game TicTacToe.


The test cases are attached in main methods as below:

TicTacToeTest game = new TicTacToeTest();
game.move(1, 1, 1);
game.move(1, 0, 2);
game.move(2, 0, 1);
game.move(0, 2, 2);
game.move(2, 2, 1);
game.move(2, 1, 2);
game.move(0, 0, 1);

game.refresh();

game.move(0, 0, 1);
game.move(1, 1, 2);
game.move(1, 0, 1);
game.move(2, 0, 2);
game.move(0, 2, 1);
game.move(2, 1, 2);
game.move(2, 2, 1);
game.move(1, 2, 2);
